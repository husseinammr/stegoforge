## 🎯 الميزات الرئيسية

### 🔐 **إخفاء النص في الصور (LSB Steganography)**
```
تقنية إخفاء البيانات بتعديل آخر Bit في كل pixel
يجعل الصورة تبدو طبيعية تماماً — الفرق في الألوان أقل من 1/255
```
- ✅ دعم صيغ: PNG, BMP, JPG, TIFF
- ✅ تشفير اختياري بكلمة مرور (XOR Cipher)
- ✅ سعة إخفاء كبيرة: حتى `width × height × 3 ÷ 8` بايت
- ✅ واجهة Drag & Drop سهلة الاستخدام

### 📤 **استخراج النص المخفي**
- ✅ استخراج فوري بدون فقدان البيانات
- ✅ دعم كلمات المرور المشفرة
- ✅ كشف تلقائي للبيانات المخفية
- ✅ نسخ سهل بزر واحد

### 🧪 **تحليل الصور الجنائي**
- 📊 معلومات الصورة الكاملة (أبعاد، نوع، حجم)
- 📈 **رسم بياني Histogram** لقنوات RGB
- 🔍 **كشف LSB Analysis** — اكتشف البيانات المخفية تلقائياً
- 📉 إحصاءات متقدمة (متوسط الألوان، نسب البتات)

### 🔑 **Hash وسلامة الملف**
- ✅ حساب: **MD5, SHA-1, SHA-256, SHA-512**
- ✅ التحقق من سلامة الملفات
- ✅ كشف التعديلات على الملفات
- ✅ مقارنة فورية للـ Hashes

## 🚀 الاستخدام السريع

### **الطريقة 1️⃣ - مباشر على الويب**
```
1. افتح steganography-toolkit.html في المتصفح
2. اختر Tab المطلوب (إخفاء/استخراج/تحليل/Hash)
3. ارفع الصورة (Drag & Drop أو اضغط)
4. اكتب النص أو أدخل البيانات
5. اضغط الزر وحمل النتيجة
```

## 📖 دليل التفصيلي

### **1️⃣ إخفاء نص داخل صورة**

```
الخطوات:
1. اذهب إلى Tab "إخفاء نص"
2. ارفع صورة (PNG أو BMP مفضل)
3. اكتب النص السري في المربع
4. (اختياري) ضع كلمة مرور للتشفير
5. اضغط "🔐 إخفاء النص في الصورة"
6. حمل الصورة الجديدة

ملاحظات:
- الصورة الناتجة تبدو طبيعية تماماً
- لا يمكن رؤية النص بالعين المجردة
- كلمة المرور تجعل الاستخراج مستحيل بدونها
```

### **2️⃣ استخراج نص مخفي**

```
الخطوات:
1. اذهب إلى Tab "استخراج نص"
2. ارفع الصورة التي تشك فيها
3. إذا كانت محمية بكلمة مرور، أدخلها
4. اضغط "🔓 استخراج النص المخفي"
5. انسخ النص المستخرج

ملاحظات:
- إذا كانت كلمة المرور خاطئة لن تحصل على شيء
- الأداة تكشف تلقائياً وجود بيانات مخفية
```

### **3️⃣ تحليل صورة جنائياً**

```
الخطوات:
1. اذهب إلى Tab "تحليل صورة"
2. ارفع أي صورة للتحليل
3. اضغط "🔬 تحليل الصورة"

النتائج:
- معلومات الصورة الكاملة
- Histogram الألوان (تحليل التوزيع)
- LSB Analysis (كشف الإخفاء المحتمل)
- إحصاءات متقدمة

تفسير LSB Analysis:
- 50% طبيعي = لا توجد بيانات مخفية
- قريب من 50% جداً = يُشتبه بوجود إخفاء
```

### **4️⃣ حساب Hashes**

```
الخطوات:
1. اذهب إلى Tab "Hash"
2. ارفع أي ملف
3. اضغط "🔑 احسب الـ Hashes"
4. انتظر النتائج (MD5, SHA-1, SHA-256, SHA-512)
5. الصق Hash متوقع للمقارنة (اختياري)

الاستخدام:
- التحقق من سلامة الملفات المحملة
- اكتشاف التعديلات على الملفات
- مطابقة الملفات المريبة من قواعد بيانات الـ Malware
```

## 🔐 الأمان والخصوصية

### **لماذا آمنة؟**
- ✅ **100% Client-Side** — كل المعالجة تتم على جهازك
- ✅ **لا توجد بيانات تُرسل للإنترنت** — يعني خصوصيتك محمية
- ✅ **بدون سجلات** — لا توجد سجلات على خادم
- ✅ **مفتوحة المصدر** — راجع الكود بنفسك
- ✅ **لا توجد ملفات مؤقتة** — كل شيء يُحذف بعد الانتهاء

### **تشفير البيانات**
```
عند استخدام كلمة مرور:
- النص الأصلي + كلمة المرور
- تمر عبر XOR Cipher
- النتيجة تُخفى في LSB
- لا يمكن الاستخراج بدون كلمة المرور الصحيحة
```
